# Awesome shadcn/ui Components

A collection of modern UI components built with shadcn/ui, Tailwind CSS and React.

## Components

* Navbar
* Hero Section
* Pricing Table
* Dashboard Sidebar
* Login Form
* Feature Sections
* Testimonials

## Tech Stack

* React
* Next.js
* Tailwind CSS
* shadcn/ui

## Usage

Copy the component you want and paste it into your project.

## Goal

To provide ready-to-use shadcn/ui components for developers building modern web apps.
